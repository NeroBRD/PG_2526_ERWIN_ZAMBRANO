⚙️ Temas y Diseño

Usa el engranaje superior para colores. Presiona las flechas ▲/▼ en cada panel para reordenar los controles a tu gusto.

🤖 Modo Macros

1. Toca 🔴 Grabar.
2. Haz movimientos.
3. Toca ⏹️ Parar.
4. Toca ▶️ Rutina para que la grúa los repita sola.

📱 Modo Inmersivo

Sostén el celular acostado. Al activarlo, inclina hacia adelante/atrás para mover el carro, y gira a los lados para rotar la grúa.

⚠️ ¿Sensor Bloqueado?
Si dice "Datos Nulos", tu navegador apagó el sensor por seguridad. Para arreglarlo:
1. Ve a chrome://flags (u opera://flags).
2. Busca Insecure origins treated as secure.
3. Escribe la IP de esta web y pon Enabled.
4. Toca Relaunch (Reiniciar navegador).
